** aux_math:

sgn(p/q,s) =  1 if 1/2 > p/q > 0 and s > 0
sgn(p/q,s) =  0 if 1/2 = p/q  OR  p/q = 0  OR  s = 0
sgn(p/q,s) = -1 if (1/2 > p/q > 0  and s < 0)  OR  (0 > p/q > -1/2  and s > 0)

To compute it, define
    sgn(s) = 1 / 0 / -1
  sgn(p/q) = 1 / 0 / -1
and multiply these together.
